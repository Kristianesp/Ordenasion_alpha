# Plan: Reestructuración UI Definitiva - Ordenasion Alpha v3.0.2

## Análisis de Problemas UI (basado en captura real)

### Problemas Críticos Visibles en la Captura

#### 1. GroupBox Titles Truncados
**Evidencia:** Los títulos "Carpeta de Origen", "Opciones de Análisis", "Resumen de Selección" se cortan parcialmente.
**Causa:** CSS `QGroupBox::title` tiene `left: 14px` y `padding: 4px 10px` que no dejan espacio suficiente para textos largos. El título se renderiza dentro del borde del GroupBox y el texto overflow no se maneja.
**Archivos:** `src/utils/modern_styles.py:73-74`

#### 2. Header Principal Truncado
**Evidencia:** "Organizador de Archivos y Carpetas" se corta en la parte superior.
**Causa:** QLabel sin `setWordWrap(True)`, sin `setSizePolicy` expansivo, y el layout header no distribuye espacio correctamente entre el título y el botón Config.
**Archivos:** `src/gui/main_window.py:177-184`

#### 3. FilterBar Apretado y Texto Truncado
**Evidencia:** 
- Placeholder "Buscar por nombre..." apenas visible
- "Todas las categorias" en el QComboBox se trunca
- Botón "Limpiar" pegado al combo sin espacio
**Causa:** 
- `filter_bar.py:22` tiene `layout.setContentsMargins(0, 5, 0, 5)` — margen izquierdo 0
- `layout.setSpacing(10)` es insuficiente para 5 elementos en una fila
- El QLineEdit no tiene `setMinimumWidth` ni `QSizePolicy` adecuado
- El QComboBox no tiene ancho mínimo suficiente
**Archivos:** `src/gui/filter_bar.py:20-60`

#### 4. Sin Márgenes Internos en GroupBox
**Evidencia:** El contenido de los GroupBox está pegado a los bordes laterales.
**Causa:** 
- `main_window.py:189` `source_layout.setContentsMargins(12, 16, 12, 12)` — los 12px son insuficientes con el border-radius de 12px del CSS
- El CSS `QGroupBox` tiene `padding-top: 20px` pero no padding lateral
**Archivos:** `src/gui/main_window.py:189, 208, 288`

#### 5. Tabla sin Espacio / Headers Pegados
**Evidencia:** Los headers de columna "Elemento", "Destino", "%", "Archivos", "Tamaño" están pegados al borde superior del GroupBox.
**Causa:** No hay separador visual entre el FilterBar y la tabla. La tabla tiene `border-radius: 12px` en CSS pero el QHeaderView no tiene margen superior.
**Archivos:** `src/gui/main_window.py:252-280`, `src/utils/modern_styles.py:92-97`

#### 6. Resumen de Selección Apretado
**Evidencia:** 
- Labels "0/0 seleccionados", "0 B", "0 archivos" se superponen
- Botones "Todo" y "Nada" son demasiado anchos para su texto
- El botón "Organizar Archivos" es enorme y desproporcionado
**Causa:** 
- QHBoxLayout sin espaciado adecuado (`summary_layout` usa default spacing)
- Labels sin `setMinimumWidth` ni `setSizePolicy`
- Botón "Organizar Archivos" con `setMinimumWidth(180)` pero el texto "Organizar Archivos" + emoji ocupa más
**Archivos:** `src/gui/main_window.py:286-320`

#### 7. Botón Config Aislado
**Evidencia:** El botón "⚙️ Config" está flotando arriba a la derecha sin relación visual con el contenido.
**Causa:** Está en un QHBoxLayout con `addStretch()` que lo empuja a la derecha, pero sin agrupación visual con el header.
**Archivos:** `src/gui/main_window.py:176-185`

#### 8. Spinbox de Similitud Roto
**Evidencia:** El spinbox muestra "7^" en lugar de "70%" con flechas up/down.
**Causa:** El CSS de QSpinBox en `modern_styles.py:109` no estiliza correctamente los botones de incremento/decremento. Los botones nativos de Windows se superponen con el estilo custom.
**Archivos:** `src/utils/modern_styles.py:109-110`

## Plan de Implementación Definitivo

### FASE 1: Arreglo de CSS Global (`modern_styles.py`) — PRIORIDAD CRÍTICA

#### 1.1 Arreglar QGroupBox::title para textos largos
**Archivo:** `src/utils/modern_styles.py`

**Problema:** El título se trunca porque el padding es insuficiente.
**Solución:**
```css
QGroupBox {
    margin-top: 12px !important;
    padding: 24px 16px 12px 16px !important;
    /* padding-top mayor para acomodar el título */
}
QGroupBox::title {
    subcontrol-origin: margin !important;
    subcontrol-position: top left !important;
    left: 12px !important;
    top: -2px !important;
    padding: 2px 14px !important;
    /* padding horizontal mayor para textos largos */
    border-radius: 8px !important;
    max-width: 80% !important;
    /* evitar que el título ocupe todo el ancho */
}
```

#### 1.2 Arreglar QSpinBox botones up/down
**Archivo:** `src/utils/modern_styles.py`

**Problema:** Los botones nativos se superponen con el estilo custom.
**Solución:** Agregar estilos explícitos para sub-controls del QSpinBox:
```css
QSpinBox {
    /* ... existente ... */
    padding-right: 20px !important;
    /* espacio para los botones */
}
QSpinBox::up-button {
    subcontrol-origin: border !important;
    subcontrol-position: top right !important;
    width: 18px !important;
    height: 18px !important;
    border: none !important;
    border-radius: 4px !important;
    background-color: transparent !important;
}
QSpinBox::down-button {
    subcontrol-origin: border !important;
    subcontrol-position: bottom right !important;
    width: 18px !important;
    height: 18px !important;
    border: none !important;
    border-radius: 4px !important;
    background-color: transparent !important;
}
QSpinBox::up-arrow {
    image: none !important;
    /* dejar que Qt renderice las flechas nativas */
}
QSpinBox::down-arrow {
    image: none !important;
}
```

#### 1.3 Agregar espaciado global a QTableView
**Archivo:** `src/utils/modern_styles.py`

**Problema:** La tabla está pegada a los bordes.

|---------|---------|---------|

## Reglas de Diseño Aplicadas

1. **Márgenes consistentes:** 16px lateral en GroupBox, 8px en FilterBar
2. **Anchos mínimos:** Cada control tiene `setMinimumWidth` apropiado
3. **Spacing adecuado:** 8-16px entre elementos relacionados
4. **Textos truncados:** Usar `setWordWrap(True)` en labels largos
5. **Botones compactos:** Width fijo cuando el texto es corto, mínimo cuando es variable
6. **Separadores visuales:** QFrame HLine entre secciones lógicas
7. **2 filas en GroupBoxes:** Evitar layouts complejos con muchas columnas
