# Changelog v3.1.0

## Added
- Nueva version de release alineada con el flujo real del proyecto.
- Empaquetado preparado para un `.exe` reproducible con PyInstaller.

## Improved
- Interfaz reorganizada en pestañas para un flujo de trabajo mas claro.
- Inicio mas rapido gracias a splash progresivo, cache de temas y carga lazy.
- Refactorizacion de estado global, workers y componentes de UI.
- Mejoras en el centro de tareas, preview, resumen y dashboard de duplicados.

## Fixed
- Versiones inconsistentes en UI y documentacion.
- Preparacion para bundling de `smartctl.exe` y ejecucion en Windows.
