Fingerprint perceptual para audio

- Coloca `fpcalc.exe` en `bin/` para que el `.exe` final pueda usar Chromaprint.
- La app intentara usar `bin/fpcalc.exe` automaticamente cuando este empaquetada.
- La clave de `AcoustID` se configura desde `Configuracion -> Biblioteca musical`.
- MusicBrainz no necesita API key.
